# Система продаж цифровых услуг

Локальная интерактивная версия курса. Интернет и установка пакетов не нужны.

## Запуск на macOS

1. Дважды нажмите `start.command`.
2. Сайт откроется по адресу `http://127.0.0.1:8080`.
3. Чтобы остановить сайт, закройте окно Terminal или нажмите `Control + C`.

Если macOS не разрешает двойной клик, откройте Terminal в этой папке и выполните:

```sh
./start.command
```

Прогресс и текущий шаг сохраняются только в локальном хранилище браузера.
